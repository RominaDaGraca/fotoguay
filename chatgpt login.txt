Aquí tienes un ejemplo básico de código PHP para hacer login utilizando una base de datos MySQL:

// Conexión a la base de datos
$servername = "localhost";
$username = "tu_usuario";
$password = "tu_contraseña";
$dbname = "tu_basededatos";

$conn = mysqli_connect($servername, $username, $password, $dbname);

// Verificación de la conexión
if (!$conn) {
  die("Connection failed: " . mysqli_connect_error());
}

// Comprobación del envío del formulario
if (isset($_POST['submit'])) {
  // Recuperación de los datos del formulario
  $username = $_POST['username'];
  $password = $_POST['password'];

  // Consulta para verificar si el usuario y la contraseña coinciden
  $sql = "SELECT * FROM users WHERE username='$username' AND password='$password'";
  $result = mysqli_query($conn, $sql);

  // Comprobación del resultado de la consulta
  if (mysqli_num_rows($result) == 1) {
    // Inicio de sesión exitoso
    session_start();
    $_SESSION['username'] = $username;
    header("Location: index.php"); // Redirige a la página principal
    exit();
  } else {
    // Inicio de sesión fallido
    echo "Usuario o contraseña incorrectos";
  }
}

// Cierre de la conexión a la base de datos
mysqli_close($conn);




Este código se encarga de conectarse a la base de datos, comprobar si se ha enviado el formulario de login, recuperar los datos del formulario, consultar la base de datos para verificar si el usuario y la contraseña coinciden, iniciar sesión si la consulta devuelve un resultado y redirigir al usuario a la página principal, y mostrar un mensaje de error si el inicio de sesión falla. Este es solo un ejemplo básico, y dependiendo de tus necesidades específicas, es posible que necesites agregar más funcionalidades y seguridad a tu código.
